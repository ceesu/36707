Visual inference

Well be doing visual inference. This means well generate random plots and hide the real diagnostic plot among them. This helps you learn what perfectly normal random variation looks like, and attunes you to the different kinds of model misspecification that might happen.

Youll need to install the nullabor package for this, as well as ggplot2.

library(nullabor)
library(ggplot2)

QQ plots

It can be hard to tell if a Q-Q plot is really normal. Here well make a lineup where one dataset follows a non-normal distribution, and the others are normal can you spot the non-normal distribution?

N <- 200 # try different sample sizes
data <- data.frame(x = rt(N, 3)) # replace rt with any other distribution

# don't change the below
d <- lineup(null_dist("x", "norm"), data)
## decrypt("fEo5 4696 Yx UnLY9Ynx zQ")
ggplot(d, aes(sample = x)) + 
  geom_qq() +
  geom_qq_line() +
  facet_wrap(~ .sample) +
  theme_bw()
